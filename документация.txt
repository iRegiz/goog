Для работа надо через терминал подключить и активировать Selenium с MySQL Conneciton

Код заработает не с первой попытки, всегда при первом запуска логики бросается ошибка, надо просто во второй раз запустить логику

Но ещё, иногда из-за высокой скорости интернета сайт может неполнастью отоброзится, итого ещё одни ошибки(задержку хоть на минуту ставь всё ровно не успеват), происходит это раз в 7-8 запуске, если случлось, просто перезапускаем

Результаты многих действии выводятся на терминале

Само структура:
Открыватся новое окно на рабочем столе с загружаешемся результатом ссылки в пременной 'url_address'(в данный момент захардкоден на ТЦ Казыну)
Координаты объекта получаем прямо из ссылки на объект пользуясь методами из библитеки 're'
В остальные и последуеших частях кода используется библиотека Selenium

Перед главным циклом (который который будет проводить главные оперции) аутециврую 4 массива(листа) для сохранение и пропускать повторяюшиеся данные


После, начинается Main Cycle

С начало подключение в БД, от-туда берём нужные данные и сохраняем 
После While цикл, в который будет парсить отзывы до конца блока страницы
Первым делам проверяем есть ли в БД автор пользвателя
Нет - добавляем и продолжаем парсить
Есть - пропускаем отзыв
После проверяем есть этот же отзыв в БД
Нет - добавляем и парсям данные
Парсили ли его до этого:
Если да то пропускаем
Нет - парсям дальше
Вывод на терминале для удобства

И так все открытые отзывы
Код скролит дальше в низ получая новые отзывы и всё по циклу

PS:Ужасно не оптимизированный код с кучай мусора, аж сам запуталься
